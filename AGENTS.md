# SafeMoto AI Agents Configuration

## Project Overview
SafeMoto is a community ride safety application for Ghana. This is a life-safety application where code quality, reliability, and security are non-negotiable.

## Technology Stack
- **Frontend**: React + Vite
- **Backend**: Supabase (PostgreSQL + Auth + Storage)
- **Styling**: Tailwind CSS
- **Platform**: Mobile-first responsive design
- **Language**: JavaScript (no TypeScript)

## Development Guidelines

### Code Quality Standards
- All code must be production-ready and thoroughly tested
- Follow mobile-first responsive design principles
- Implement proper error handling and user feedback
- Use semantic HTML5 elements for accessibility
- Optimize for performance and battery life on mobile devices

### Security Requirements
- Implement proper authentication and authorization
- Validate all user inputs
- Use HTTPS for all communications
- Implement rate limiting for API calls
- Follow OWASP security guidelines

### Database Best Practices
- Use proper indexing for performance
- Implement row-level security (RLS) policies
- Use transactions for data consistency
- Optimize queries for mobile network conditions
- Implement proper backup strategies

## Agent Responsibilities

### Development Agent
- Write clean, maintainable JavaScript code
- Follow React best practices and hooks
- Implement responsive UI components
- Ensure cross-browser compatibility
- Optimize for mobile performance

### Security Agent
- Review code for security vulnerabilities
- Implement proper authentication flows
- Validate and sanitize all inputs
- Ensure data privacy and protection
- Monitor for security threats

### Database Agent
- Design efficient database schemas
- Implement proper RLS policies
- Optimize query performance
- Ensure data integrity
- Handle database migrations safely

### Testing Agent
- Write comprehensive unit tests
- Implement integration tests
- Perform end-to-end testing
- Test on various mobile devices
- Ensure accessibility compliance

## Emergency Protocols
This is a life-safety application. Any bugs or issues that could affect user safety must be treated as critical priority and resolved immediately.

## Contact Information
For urgent safety-related issues, contact the development team immediately through the emergency communication channel.
